# Структура веб проекта:

- src/ — исходные файлы проекта
- src/components/ — папка с JS компонентами
- src/components/scss/ — папка с со стилями
- src/components/scss/components — папка с файлами компонентов стилей для блоков на веб старице

- index.html — HTML-файл главной страницы• info.html — HTML-файл с дополнительной информацией и обратной связи страницы

# Интернет-магазин "Web coffee storе"

"Web coffee storе" — интернет-магазин для любителей кофе , где пользователи могут просматривать товарыа а любимые напитки добавлять в корзину и оформлять заказы оплата которых может всключать онлайн или при получении . Сайт предоставляет удобный интерфейс просмотр деталей товаров, управления корзиной и выбора способа оплаты, обеспечивая полный цикл покупки с отправкой заказов на сервер.




# Сервер приложения

Запуск сервера

```
node server.cjs
```

1. Назначение сервера
   Этот сервер выполняет две ключевые функции:

- Отдача статического фронтенда (файлы из папки dist/).
- Обработка API‑запросов (маршрут /api/subscribe для приёма данных формы).

2. Зависимости
   Проект использует:

- express — веб‑сервер;
- cors — управление CORS‑политикой;
- встроенные модули Node.js (path).

3. Режимы работы

- Запустите Vite (фронтенд): Для разработки (npm run dev)

```
bash
npm run dev  # запускает Vite на localhost:5173
node server.cjs  # запускает API на localhost:3000
Открывайте http://localhost:5173 (фронтенд).
```

Запросы к /api/subscribe автоматически проксируются на порт 3000.

- Для продакшена

```
bash
npm run build  # собирает dist/
node server.cjs  # запускает сервер
Открывайте http://localhost:3000 (и фронтенд, и API доступны здесь).
```

- Команды

```
bash
# Сборка фронтенда
npm run build

# Запуск сервера
node server.cjs

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```
